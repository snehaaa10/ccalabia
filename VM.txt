Task 1: Virtual Machines (VMs) CCA Lab setup

Phase 1: Installation & Initial Setup:
1.	Open File Explorer: Open a new volume in File Explorer.
2.	Locate Folder: Go to the Ubuntu NS2 folder.
3.	Launch VirtualBox: Click on the VirtualBox executable.
4.	Security: Enter the password: 1admin*bnm
5.	Install: Complete the installation process.
6.	Open App: Open the Oracle VM VirtualBox application.

Phase 2: Creating the Virtual Machine:
1.	New VM: Click on the "New" button.
2.	Name & OS: Give the VM a name and click Next.
3.	Hardware Configuration:
• Base Memory: Set to above 5000 MB.
• Processors: Set to 3.
• Click Next.
4.	Hard Disk Setup: Click on "Use an existing Virtual Hard Disk".
• Click the Folder icon to browse.
• Click the Add symbol.
• Go to the Ubuntu NS2 folder.
• Select the file "ubuntunsS010".
• Click Choose, then click Next.
5.	Finish: Click the Finish button to create the VM.

Phase 3: Settings & Execution:
1.	Settings: Click on the Settings icon for your new VM.
2.	Display Settings: Go to the Display tab and configure the following:
• Video Memory: Above 90 MB.
• Monitor Count: 3 or 4.
• Scale Factor: 150%.
• Enable 3D Acceleration: Check this box.
3.	Confirm: Click OK.
4.	Launch: Click Start to run the Virtual Machine.

Phase 4:Execute a program
Program:
#include <stdio.h>
void main() {
    printf("Hello, World!\n");
}
to get output: ./a.out
